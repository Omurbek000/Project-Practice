# График работы сотрудников (12 человек / 4 смены)

Простое веб-приложение для составления недельного графика работы 12 сотрудников с выбором одной из 4 смен или выходного дня. Данные сохраняются в браузере (localStorage).

![Превью](screenshot.png) *(если добавите скриншот)*

## 🚀 Возможности

- **12 сотрудников** (имена можно менять прямо в коде).
- **4 смены** + выходной.
- Выбор **недели** (любая дата начала → автоматически 7 дней).
- Удобная таблица: сотрудник × день недели.
- Сохранение / загрузка графика в **localStorage** браузера.
- Сброс всех назначений на «Выходной».
- Адаптивная вёрстка: работает на компьютерах и планшетах (горизонтальная прокрутка).

## 🛠️ Технологии

- HTML5
- CSS3 (flex, grid, медиазапросы)
- Чистый JavaScript (без зависимостей)

## 📦 Установка и запуск

### Способ 1 — локально

1. Скачайте файл `grafik.html` (или создайте его, скопировав код).
2. Откройте файл любым браузером (Chrome, Firefox, Edge, Opera).
3. Работайте с графиком — все изменения автоматически сохранятся в браузере.

### Способ 2 — через редактор кода

Если хотите изменять код (например, добавить больше сотрудников или сменить названия смен):

1. Откройте `grafik.html` в VS Code, PyCharm или даже Блокноте.
2. Найдите в начале скрипта переменные:
   ```js

   const DEFAULT_EMPLOYEES = [ "Анна К.", "Борис Л.", ... ];
   const SHIFTS = [
       { value: "shift1", label: "🚀 Смена 1" },
       ...
   ];



   Измените значения по своему усмотрению.

Сохраните файл и обновите страницу в браузере.

🧑‍💻 Как пользоваться
Выберите дату начала недели — таблица автоматически покажет 7 дней (пн–вс).

Для каждого сотрудника в каждый день выберите из выпадающего списка: Смена 1, Смена 2, Смена 3, Смена 4 или Выходной.

Нажмите «Сохранить график» — данные запомнятся в браузере.

При следующем открытии страницы нажмите «Загрузить последний» (или данные подгрузятся автоматически при первом запуске).

Кнопка «Сбросить всё» возвращает всех на Выходной и восстанавливает имена по умолчанию.

💡 При изменении дат начала недели смены переносятся на те же числа календаря, если они есть в новой неделе.

📁 Структура файла
Весь проект — один файл grafik.html, который содержит:

стили (в <style>),

разметку (таблица, кнопки),

скрипт (логика генерации, сохранения, рендеринга).

Никаких внешних зависимостей, серверов или сборщиков.

];


Убрать автосохранение
Закомментируйте вызов autoSaveToLocal() в обработчике change селектов.

📝 Примечания
Все данные хранятся только в localStorage вашего браузера. Если вы очистите историю / кэш сайтов, график может быть утерян. Для надёжности используйте кнопку «Сохранить график» и не чистите данные.

Приложение не отправляет данные в интернет — полностью офлайн после загрузки.
